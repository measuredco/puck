# resolveAllData

Utility function to execute all [`resolveData` methods](/docs/api-reference/component-config#resolvedatadata-params) on a data payload outside of the [`<Puck>`](/docs/api-reference/components/puck) editor, returning the updated value.

```tsx copy
import { resolveAllData } from "@measured/puck";

const updatedData = await resolveAllData(data, config);
```

This is useful if you need to run your resolvers before passing your data to [`<Render>`](/docs/api-reference/components/render).

## Args

| Param    | Example                     | Type                                 |
| -------- | --------------------------- | ------------------------------------ |
| `data`   | `{ content: {}, root: {} }` | [Data](/docs/api-reference/data)     |
| `config` | `{ components: {} }`        | [Config](/docs/api-reference/config) |

## Returns

The updated [Data](/docs/api-reference/data) object.
