# Custom

Implement a field with a custom UI. Extends [Base](base).

<ConfigPreview
  label="Example"
  componentConfig={{
    fields: {
      title: {
        type: "custom",
        render: ({ name, onChange, value }) => {
          return (
            <FieldLabel label="Title">
              <input
                defaultValue={value}
                name={name}
                onChange={(e) => onChange(e.currentTarget.value)}
                style={{ border: "1px solid hotpink", padding: 4 }}
              />
            </FieldLabel>
          );
        },
      },
    },
    defaultProps: {
      title: "Hello, world",
    },
    render: ({ title }) => {
      return <p style={{ margin: 0 }}>{title}</p>;
    },
  }}
/>

```tsx {3-12, 18-21} copy
import { FieldLabel } from "@measured/puck";

const render = ({ name, onChange, value }) => (
  // You'll normally want to wrap with FieldLabel to get the default field label styles
  <FieldLabel label="Title">
    <input
      defaultValue={value}
      name={name}
      onChange={(e) => onChange(e.currentTarget.value)}
    />
  </FieldLabel>
);

const config = {
  components: {
    Example: {
      fields: {
        title: {
          type: "custom",
          render,
        },
      },
      render: ({ title }) => {
        return <p>{title}</p>;
      },
    },
  },
};
```

## Params

| Param                       | Example                   | Type     | Status   |
| --------------------------- | ------------------------- | -------- | -------- |
| [`type`](#type)             | `type: "custom"`          | "custom" | Required |
| [`render()`](#renderparams) | `render: () => <input />` | Function | Required |

## Required params

### `type`

The type of the field. Must be `"custom"` for Custom fields.

```tsx {6} copy
const config = {
  components: {
    Example: {
      fields: {
        title: {
          type: "custom",
          render: CustomField,
        },
      },
      // ...
    },
  },
};
```

### `render(params)`

Render the custom field. It's normally best to wrap your custom input in the [`<FieldLabel>` component](/docs/api-reference/components/FieldLabel) to get the default field label styling.

```tsx {3-11,19} copy
import { FieldLabel } from "@measured/puck";

const render = ({ name, onChange, value }) => (
  <FieldLabel label="Title">
    <input
      defaultValue={value}
      name={name}
      onChange={(e) => onChange(e.currentTarget.value)}
    />
  </FieldLabel>
);

const config = {
  components: {
    Example: {
      fields: {
        title: {
          type: "custom",
          render,
        },
      },
      // ...
    },
  },
};
```

#### `params`

| Param             | Example                    | Type     |
| ----------------- | -------------------------- | -------- |
| `name`            | `"title"`                  | String   |
| `onChange(value)` | `onChange("Hello, world")` | Function |
| `value`           | `"Hello, world"`           | Any      |
