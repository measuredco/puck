# Base

The base type shared by all fields.

## Params

| Prop              | Example          | Type   | Status |
| ----------------- | ---------------- | ------ | ------ |
| [`label`](#label) | `label: "Title"` | String | -      |

## Optional params

### `label`

A label for the input. Puck will use the key if not provided.

```tsx {6} copy
const config = {
  components: {
    Example: {
      fields: {
        items: {
          label: "My Field",
          // ...
        },
      },
      // ...
    },
  },
};
```
