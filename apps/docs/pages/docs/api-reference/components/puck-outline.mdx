# \<Puck.Outline\>

Render an interactive outline of the current data payload when composing a custom Puck UI.

```tsx {5} copy
import { Puck } from "@measured/puck";

export function Editor() {
  return <Puck {/*...*/}>
    <Puck.Outline />
  </Puck>;
}
```

## Props

This component doesn't accept any props.
