# Metadata

A global data object provided to Puck for use in `resolveData` and `render`.

```json copy
{
  "title": "My cool page title",
  "content": "Lorem ipsum dolor sit amet, consectetur adipiscing elit...",
  ...
}
```

## Params

| Param         | Example       | Type | Status   |
| ------------- | ------------- | ---- | -------- |
| [key: string] | `content: []` | Any  | Optional |

### `resolveData`

Metadata can be used during within the resolveData API to apply mutations to the props. Refer to [](/docs/api-reference/configuration/component-config#paramsmetadata) for more information.

### `render`

Metadata can be used during within the render API to alter the final output of the component. Usage here impacts only the rendered output and does not mutate the Puck data. Refer to [](/docs/api-reference/configuration/component-config#puckmetadata) for more information.
