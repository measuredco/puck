# fields

Override the fields wrapper.

```tsx copy
const overrides = {
  fields: ({ children }) => <div>{children}</div>,
};
```

## Props

| Prop                    | Example   | Type      |
| ----------------------- | --------- | --------- |
| [`children`](#children) | `<div />` | ReactNode |

### `children`

The default node for the fields.
