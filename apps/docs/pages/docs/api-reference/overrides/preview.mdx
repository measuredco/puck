# preview

Override the drag-and-drop preview.

```tsx copy
const overrides = {
  preview: ({ children }) => <div>{children}</div>,
};
```

## Props

| Prop                    | Example   | Type      |
| ----------------------- | --------- | --------- |
| [`children`](#children) | `<div />` | ReactNode |

### `children`

The default node for the preview.
