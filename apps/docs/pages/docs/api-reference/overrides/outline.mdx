# outline

Override the outline.

```tsx copy
const overrides = {
  outline: ({ children }) => <div>{children}</div>,
};
```

## Props

| Prop                    | Example   | Type      |
| ----------------------- | --------- | --------- |
| [`children`](#children) | `<div />` | ReactNode |

### `children`

The default node for the outline.
