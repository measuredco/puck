# header

Override the header.

```tsx copy
const overrides = {
  header: ({ actions }) => (
    <header>
      <span>My header</span>
      <div>{actions}</div>
    </header>
  ),
};
```

## Props

| Prop                    | Example   | Type      |
| ----------------------- | --------- | --------- |
| [`actions`](#actions)   | `<div />` | ReactNode |
| [`children`](#children) | `<div />` | ReactNode |

### `actions`

A node containing the [`headerActions`](header-actions).

### `children`

The default node for the header.
