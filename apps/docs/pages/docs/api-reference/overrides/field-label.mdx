# fieldLabel

Override the label for all internal fields.

```tsx copy
const overrides = {
  fieldLabel: ({ children, label }) => (
    <label>
      <div>{label}</div>
      {children}
    </label>
  ),
};
```

See [FieldLabel](/docs/api-reference/components/field-label) component.
